# WatchParty

Serverless synchronized video watching. Create a room, share the link, paste a video URL, watch together.

## Stack
- Next.js (Vercel)
- PartyKit (serverless WebSocket rooms)
- react-player (YouTube)
- HLS.js (streaming sites)
- cheerio (video source extraction)

## Features
- Room creation with shareable links
- Private rooms (PIN-protected)
- Synced play/pause/seek across all viewers
- YouTube + streaming site support
- No auth required
